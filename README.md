# @hooksvue/big-screen 文档站

这个仓库使用 VitePress 编写 `@hooksvue/big-screen` 的中文文档。

## 本地开发

```bash
pnpm install
pnpm docs:dev
```

## 构建

```bash
pnpm docs:build
pnpm docs:preview
```
